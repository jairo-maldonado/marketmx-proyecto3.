# MarketMX - Agencia de Marketing Digital

Proyecto de Landing Page desarrollada para la asignatura de Desarrollo Web Avanzado. Es un sitio corporativo responsivo para una agencia que ofrece servicios de SEO, SEM y redes sociales.

## Tecnologías Utilizadas
- React 18+ (Vite)
- Bootstrap 5
- CSS Clásico

## Características Principales
- Navbar responsivo con menú hamburguesa.
- Hero section con imagen de fondo y CTA.
- Modales dinámicos para el detalle de los servicios.
- Acordeón interactivo para explicar el proceso de trabajo.
- Formulario de contacto con validación de datos.

## Instrucciones de Instalación
Para correr este proyecto en un entorno local (como en el computador de la presentación), sigue estos pasos:

1. Clona este repositorio en tu máquina local.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el comando `npm install` para descargar todas las dependencias (node_modules).
4. Ejecuta el comando `npm run dev` para iniciar el servidor local.
5. Abre tu navegador en la dirección indicada en la terminal (usualmente http://localhost:5173).
