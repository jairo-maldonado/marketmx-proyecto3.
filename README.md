# MarketMX - Agencia de Marketing Digital

Proyecto 3: Evaluación de Desarrollo Web Avanzado (Landing Page con React + Bootstrap). 
Desarrollado para INACAP Sede Renca.

**Autor:** Jairo Maldonado

## Descripción
Sitio corporativo responsivo para una agencia que ofrece servicios de SEO, SEM y redes sociales. El proyecto está construido bajo una arquitectura de componentes funcionales e implementa gestión de estado para la interactividad.

## Tecnologías Utilizadas
* React 18+ (Vite)
* Bootstrap 5
* CSS Nativo

## Requisitos Cumplidos
* **Navbar:** Responsivo con menú hamburguesa en dispositivos móviles.
* **Hero Section:** Cabecera con imagen de fondo y Call to Action.
* **Servicios:** Modales interactivos controlados por el estado de React (`useState`).
* **Casos de Éxito:** Galería de clientes locales (App MACH, GoodMeal, Awto Chile).
* **Procesos:** Acordeón de Bootstrap 100% funcional.
* **Contacto:** Formulario con validación activa de campos vacíos.
* **Footer:** Información corporativa y ubicación (Renca, Santiago).
